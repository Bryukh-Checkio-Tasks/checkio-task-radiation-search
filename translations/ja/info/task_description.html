ロボット達は放射能が発生している島への供給を停止し、拾った最後のコンテナを調査していました。 そこには5つの異なる種類のスペアパーツが入っており種類毎に同じ番号が書かれてました。 スペアパーツの一番大きいグループから放射能が発生しているようです。 (各パーツは隣接して納めれらています) コンテナの中からそのグループを発見し、ロボット達にそのスペアパーツの数と種類を教えてあげてましょう。

コンテナは正方行列として表されます。 1から5の番号がスペアパーツの種類をあらわすラベルとして使用されており、それが行列の要素となっています。 0は空のマスを表します。 同じ番号で互いに隣接している一番大きいグループを発見し、そしてそのスペアパーツの数と番号を指摘してください。

入力: 正方行列 (リストのリスト)。 各リストはinteger型からなる。

出力: 最大のグループに含まれるスペアパーツの数と種類の番号 (2つのinteger型からなるリスト)

例:

どのように使われるか: このタスクで Union-findingアルゴリズムと素集合データ構造(Disjoint-set data structures)について学ぶことができるでしょう。 これは画像認識や地理の分析、集合の分割モデルにさえ使用されています。

事前条件:
3 ≤ len(matrix) ≤ 10
all(all(0 ≤ x ≤ 5 for x in row) for row in matrix
any(any(x for x in row) for row in matrix
テストの解答は常に1種類です。
